# Deployment Guide

Run the bot 24/7 so you can close your laptop. Several options, ranked by cost & ease.

## TL;DR

The fastest deploy: **Oracle Cloud Always Free** (truly free forever, 4 OCPU + 24GB RAM). Get it at https://cloud.oracle.com/free.

## Option 1: Oracle Cloud Always Free (Recommended — $0/month, forever)

Oracle's "Always Free" tier includes:
- 4 OCPU ARM Ampere A1
- 24 GB RAM
- 200 GB block storage
- 10 TB outbound transfer/month

**Steps:**

1. Sign up at https://cloud.oracle.com/free (requires credit card for verification, won't be charged)
2. Create a VM instance: Compute → Instances → Create Instance
   - Image: Canonical Ubuntu 22.04 (or any Linux)
   - Shape: `VM.Standard.A1.Flex` (Ampere ARM, free tier)
   - OCPU: 4, RAM: 24GB
   - Boot volume: 200GB
   - Download the SSH private key
3. Wait for instance to provision (~2 min)
4. SSH in: `ssh -i <key> ubuntu@<public-ip>`
5. Install Docker:
   ```bash
   curl -fsSL https://get.docker.com -o get-docker.sh
   sudo sh get-docker.sh
   sudo usermod -aG docker ubuntu
   newgrp docker
   ```
6. Clone + configure:
   ```bash
   git clone https://github.com/1337-ishaan/memecoin-trading-bot.git
   cd memecoin-trading-bot
   cp .env.example .env
   nano .env  # add your TELEGRAM_*, HELIUS_API_KEY (optional), BIRDEYE_API_KEY (optional)
   ```
7. Run:
   ```bash
   docker compose up -d
   docker compose logs -f
   ```

The bot will restart automatically on crashes and on system reboot. Volume `bot-data` persists your SQLite DB and PID file.

**Cost: $0/month forever** (within Always Free limits).

## Option 3: DigitalOcean (~$4-6/month)

1. Sign up at https://www.digitalocean.com
2. Basic Droplet: 1 vCPU, 1GB RAM ($4/mo) or 2 vCPU, 2GB ($6/mo) — Ubuntu 22.04
3. SSH in
4. Same Docker steps

**Cost: $4-6/month.**

## Option 4: Railway.app (easiest, ~$5/mo free credit)

1. Sign up at https://railway.app
2. Click "New Project" → "Deploy from GitHub repo"
3. Select this repo
4. Add environment variables (TELEGRAM_*, etc.) in the Variables tab
5. Railway auto-detects the Dockerfile

**Cost: $5 free credit, then ~$5/mo for the smallest instance.**

## Option 5: Fly.io (free allowance)

1. Install flyctl: `brew install flyctl` (or curl install)
2. Sign up: `fly auth signup`
3. `fly launch` in the project directory
4. Set secrets: `fly secrets set TELEGRAM_BOT_TOKEN=... TELEGRAM_CHAT_ID=...`
5. `fly deploy`

**Cost: Free for small apps (3 shared VMs, 3GB storage).**

## Option 6: Run on a spare Mac/Linux box at home

If you have any always-on machine (Mac mini, old laptop, Raspberry Pi 4+), use macOS launchd or Linux systemd.

### macOS (launchd)

Already provided: `launchd/com.memecoin-bot.plist` (generated by `npm run cli -- install-service` if you implement that).

Quick install:
```bash
cp launchd/com.memecoin-bot.plist ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.memecoin-bot.plist
launchctl start ~/Library/LaunchAgents/com.memecoin-bot
```

### Linux (systemd)

Create `/etc/systemd/system/memecoin-bot.service`:
```ini
[Unit]
Description=Memecoin Trading Bot
After=network.target

[Service]
Type=simple
User=YOUR_USER
WorkingDirectory=/path/to/memecoin-trading-bot
ExecStart=/usr/bin/env npx tsx src/cli.ts start
Restart=always
RestartSec=10
EnvironmentFile=/path/to/memecoin-trading-bot/.env

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl daemon-reload
sudo systemctl enable memecoin-bot
sudo systemctl start memecoin-bot
sudo journalctl -u memecoin-bot -f
```

## Verifying the deploy

After deploying, on the server:
```bash
docker compose ps
docker compose logs -f bot
```

You should see the bot boot banner + a 🟢 "Bot started" message in your Telegram.

## Monitoring

```bash
# Live logs
docker compose logs -f

# Last 100 lines
docker compose logs --tail=100

# Status / portfolio
docker compose exec bot npx tsx src/cli.ts status

# Recent signals
docker compose exec bot npx tsx src/cli.ts signals

# PnL breakdown
docker compose exec bot npx tsx src/cli.ts pnl

# Stop the bot
docker compose stop

# Restart
docker compose restart

# Update to latest version
git pull && docker compose up -d --build
```

## Backups

The SQLite database is in the `bot-data` volume. To back up:
```bash
docker compose exec bot sqlite3 /app/data/bot.db ".backup /app/data/backup.db"
docker compose cp bot:/app/data/backup.db ./backup-$(date +%F).db
```

Or just snapshot the volume:
```bash
docker run --rm -v memecoin-trading-bot_bot-data:/data -v $(pwd):/backup alpine \
    tar czf /backup/bot-data-$(date +%F).tgz -C /data .
```

## Security notes

- Your `.env` contains the Telegram bot token — keep it out of git
- `data/bot.db` contains trade history — back it up
- If you use Helius/Birdeye, treat the API keys as sensitive
- Never commit real wallet private keys to this repo (the bot only needs read access for KOL mirroring)
